## 常见问题处理

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 502 Bad Gateway | API 服务未启动 | `docker compose logs api` 检查 |
| SSL 证书错误 | 证书路径错误 | 确认 `ssl/fullchain.pem` 和 `ssl/privkey.pem` 存在 |
| 证书过期 | Let's Encrypt 90天过期 | `certbot renew` 续期 |
| 端口占用 | 80/443 端口被占用 | `netstat -tlnp \| grep -E ':(80\|443)'` 检查 |
| 飞书机器人无响应 | Webhook URL 不通 | 确认 HTTPS 可公网访问 |
