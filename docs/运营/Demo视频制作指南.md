## 1. 视频概述

### 1.1 视频定位

| 项目 | 说明 |
|------|------|
| **视频类型** | 产品功能 Demo 演示 |
| **目标受众** | 开发者、自动化爱好者、企业用户 |
| **视频时长** | 3~5 分钟（精简版）/ 8~10 分钟（完整版） |
| **语言** | 中文配音 + 中英文字幕 |
| **风格** | 技术演示，简洁专业 |

### 1.2 核心展示功能

根据项目功能梳理，本次 Demo 重点展示以下 6 大能力：

1. **可视化工作流设计器** — 拖拽式节点编排
2. **AI 任务节点** — OpenAI GPT 集成，变量占位符
3. **触发方式** — 手动触发 + CRON 定时触发
4. **工具节点** — 飞书消息、飞书文档、HTTP 请求
5. **条件分支** — If/Else 逻辑判断
6. **执行日志** — 实时监控、节点级日志、状态追踪

## 4. 录制计划

### 4.1 推荐工具

| 用途 | 推荐工具 | 备选 |
|------|----------|------|
| 屏幕录制 | OBS Studio（免费开源） | Camtasia、Bandicam |
| 录音 | Audacity（免费）+ 独立麦克风 | Adobe Audition |
| 视频剪辑 | DaVinci Resolve（免费版） | Premiere Pro、剪映 |
| 字幕 | 剪映（自动生成字幕） | 网易见外、Arctix |
| 配音（AI旁白） | 剪映配音 / ElevenLabs | 讯飞智作 |
| 视频压缩 | HandBrake | FFmpeg |

### 4.2 录制步骤

**Step 1｜准备环境（录制前）**
- 启动本地 Docker 服务（`docker compose up -d`）
- 浏览器打开 `http://localhost`，登录演示账号
- 准备一个"每日早报"示例工作流（提前配置好，录制时直接展示运行效果）
- 确认飞书测试群 Webhook 可用

**Step 2｜屏幕录制**
- 分段录制（每 Part 单独一个文件，方便剪辑）
- 设置分辨率：1920×1080，帧率 30fps
- 录制区域：全屏或 1280×720 窗口
- 鼠标点击添加放大效果（使用 CursorLight 等工具）

**Step 3｜配音录制**
- 方式 A（推荐）：用剪映的 AI 配音功能，读旁白文本自动生成
- 方式 B：自己录音，用 Audacity 降噪处理
- 配音语速：约 150~180 字/分钟

**Step 4｜后期剪辑（DaVinci Resolve / 剪映）**
- 拼接各段录像
- 配音/字幕对齐
- 添加转场（建议用简洁的 Dissolve 或 Cut，不花哨）
- 片头/片尾字幕卡
- 导出：H.264，1080p，码率 8~15Mbps

**Step 5｜字幕处理**
- 使用剪映自动识别字幕（识别准确率约 85%，需人工校对）
- 保留关键术语原文（如 GPT-4o、API、CRON 等）
- 可选：添加英文翻译字幕（面向国际开发者）

### 4.3 录制检查清单

```
[ ] Docker 服务运行正常
[ ] 演示账号已登录
[ ] 示例工作流已保存（每日早报）
[ ] 飞书测试群 Webhook 可用
[ ] 屏幕录制软件设置正确（1080p/30fps）
[ ] 录音设备正常工作
[ ] 无敏感信息在屏幕上（API Key 等已隐藏或打码）
[ ] 浏览器书签/标签页已整理
[ ] 背景干净整洁（建议使用纯色/模糊背景）
```

## 5. 字幕与配音方案

### 5.1 配音方案对比

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **剪映 AI 配音** | 免费、操作简单、中文自然 | 情感略显平淡 | ⭐⭐⭐⭐ |
| **ElevenLabs** | 情感丰富、可定制声音 | 需付费/注册 | ⭐⭐⭐⭐⭐ |
| **真人配音** | 最自然、有情感 | 需专业设备、耗时 | ⭐⭐⭐ |
| **无配音纯字幕** | 制作最快 | 传播效果较弱 | ⭐⭐⭐ |

**推荐**：先用剪映 AI 配音（免费快速），后续可重录 ElevenLabs 版本

### 5.2 字幕规范

| 项目 | 规范 |
|------|------|
| 字体 | 思源黑体 / 阿里巴巴普惠体 |
| 大小 | 画面高度的 1/40（如 1080p 用 28px） |
| 颜色 | 白色 #FFFFFF，带黑色描边（避免白字消失在白色背景） |
| 位置 | 画面底部 1/5 处，居中 |
| 行数 | 最多 2 行，超长字幕分帧 |
