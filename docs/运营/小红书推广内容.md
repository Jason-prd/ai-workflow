```
程序员开源了这个工具，产品经理用了都说香 👨‍💻
```

**标签：**
`#程序员 #开源项目 #GPT #FastAPI #React #自动化工具 #AI工作流 #GitHub #技术干货`

**正文：**

```
🧑‍💻 技术人，做了个让非技术也能用的工具

GitHub: github.com/Jason-prd/ai-workflow

## 这个项目是什么

AI 自动化工作流平台

▸ 后端：FastAPI + SQLAlchemy
▸ 前端：React 18 + TypeScript + React Flow
▸ 部署：Docker Compose 一行命令

## 核心功能

🤖 AI 任务节点
  - GPT-4o / GPT-4o-mini
  - {{variable}} 变量占位符
  - Temperature / Max Tokens 可调

🔔 触发器
  - 手动触发
  - CRON 定时

📤 工具节点
  - 飞书消息（卡片/文本）
  - 飞书文档（创建/读取）
  - HTTP 请求（任意 API）

🔀 条件分支
  - If / Else 逻辑

📋 执行日志
  - 节点级追踪
  - 失败自动重试

## 为什么做这个

解决自己每天写日报的痛苦 😄

不想每次都手动整理，于是用这个平台搭了一个"AI 日报生成器"，每天定时运行，自动推送到飞书群。

## 亮点

🇨🇳 国内首个飞书深度集成的开源 AI 工作流
   - 飞书消息触发（关键词/群 ID）
   - 飞书日历触发（提前提醒）
   - 执行结果卡片推送
   - 自动创建/读取飞书文档

## 快速体验

$ git clone https://github.com/Jason-prd/ai-workflow.git
$ cd backend && docker compose up -d

打开 http://localhost 即可

## 求 Star ⭐

你的支持是我继续更新的动力！

有技术问题欢迎提 Issue，代码 review 和 PR 都欢迎 🙌
```
