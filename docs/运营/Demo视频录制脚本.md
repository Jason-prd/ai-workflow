# Demo 视频录制脚本

> 目标：制作一个 3-5 分钟的产品 Demo 视频
> 用途：Twitter/X 推广、GitHub README、社交媒体展示
> 录制工具建议：OBS Studio（免费）/ Camtasia / Bandicam

---

## 视频整体结构（总时长：约 4 分钟）

| 章节 | 时长 | 内容 |
|------|------|------|
| 开场 Hook | 15秒 | 痛点引入，引出产品 |
| 产品介绍 | 30秒 | 整体功能概览 |
| Demo 演示 | 2分30秒 | 核心功能实操 |
| 结尾号召 | 15秒 | Star 邀请 + 链接 |

---

## 录制注意事项

### 环境准备
- [ ] 使用干净的浏览器界面（最好用隐身模式，避免书签/插件干扰）
- [ ] 提前准备测试数据（示例输入数据）
- [ ] 确保飞书配置已完成（如果录制飞书集成功能）
- [ ] 提前登录系统，停在工作流列表页

### 录制技巧
- [ ] 使用 1920x1080 分辨率
- [ ] 语速适中，每句话之间留 0.5 秒停顿
- [ ] 鼠标移动要慢、稳，避免快速抖动
- [ ] 可以分段录制，每段 30 秒，减少出错重录成本

### 剪辑建议
- [ ] 开场加一个 3 秒倒计时（3-2-1）
- [ ] 每个功能切换时加过渡动效
- [ ] 重要操作步骤可加**字幕高亮**
- [ ] 全局加轻量背景音乐（音量低于配音）
- [ ] 片头/片尾使用统一模板

### 推荐工具
| 工具 | 平台 | 特点 |
|------|------|------|
| OBS Studio | Win/Mac/Linux | 免费、强大、直播录制均可 |
| Camtasia | Win/Mac | 专业剪辑一体，适合 Windows |
| ScreenToGif | Win | 轻量 GIF 录制 |
| CapCut | Win/Mac | 剪辑简单，适合新手 |
